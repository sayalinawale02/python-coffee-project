# Coffee Ordering System (Python)

## 📌 Description
This is a Python-based CLI application that allows users to order coffee.

## 🚀 Features
- Select coffee type
- Add extra items (milk, sugar, etc.)
- Automatic bill calculation
- Menu-driven program

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Open terminal
2. Run:
   python CoffeeApp.py
